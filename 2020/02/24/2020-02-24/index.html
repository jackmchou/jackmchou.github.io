Process is an abstraction of node while it is running.
Process.exit(1) means the process has encountered error and exits node with code 1
process.exit(0) means process completed without error, exit process to allow for other process

What is Webpack?
Webpack is a static module bundler for modern JS apps. It builds a dependency graph that maps modules based on project needs then generate bundles

How do you add a devDependency to a package?
npm install –save-dev

What is an NPM script?
The script property of package json object, scripts to automate repetitive tasks

How do you execute Webpack with npm run?
npm run build provided webpack is specified in the npm script

What is an ES6 module and how is it different from a CommonJS module?
An ES6 module uses import and export keyword vs require to utilizing modules in Common JS

What kind of modules can Webpack support?
A lot, some of the named ones are
CoffeeScript
TypeScript
ESNext (Babel)
Sass
Less
Stylus
Elm

What is React?
A JavaScript frontend framework for building UI
What is a React element?
An html element rendered via ReactDOM package?
$$typeof is marker for react to know it’s an react object

How do you mount a React element to the DOM?
Define it using JSX or createElement method of React obj then render method of the reactDOM obj
