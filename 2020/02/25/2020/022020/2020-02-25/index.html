Quiz Q & A:
What is Babel?
JS compiler, toolchain mainly used to convert ES6 code to backwards compatible version of JS in current and older browsers or env.

What is a Plug-in?
software component that adds specific features to an existing computer program

What is a Webpack loader?
Transformation that are applied to the source code of a module. loaders parse file from other languages to JS?
Preprocessors of files as we load them?

How can you make Babel and Webpack work together?
Add them as devDependencies via npm?

What is JSX?
JSX is a syntax extension to JS dependent on the react library.
Superset of JS.

Why must the React object be imported when authoring JSX?
Because without the React library JSX is meaningless to JS

How can you make Webpack and Babel work together to convert JSX into valid JavaScript?
